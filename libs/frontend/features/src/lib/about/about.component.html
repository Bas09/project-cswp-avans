Client-side web development projectcasus


Deze webapplicatie is gemaakt in het kader van Client-Side Web Development, individueel project


In deze applicatie hebben muziekliefhebbers de mogelijkheid om een afspeellijst samen te stellen, waaraan ze hun favorieten nummers kunnen toevoegen. Deze playlists kunnen gedeeld worden met vrienden van de gebruiker.


Naast gebruikers kunnen artiesten zelf ook inloggen op de app. Artiesten kunnen hierdoor zelf nummers toevoegen en aanpassingen maken aan bestaande nummers van hun.

Datamodel / ERD

Deze applicatie heeft de entiteiten User, Playlist, Song, Artist. Friend is een zwakentiteit want deze kan zonder User niet bestaan.

De relaties zijn alsvolgt:

User - Playlist: Een user kan 0 of meerdere playlists hebben.
Playlist - user: Iedere aangemaakte playlist hoort tot 1 user.

Playlist - Song: Iedere playlist bevat 0 of meerdere songs.
Song - Playlist: Iedere song kan aan 0 of meerdere playlist toegevoegd worden.

User - Artist: Een user kan 0 of meerdere artiesten aanmaken wanner de user admin rechten heeft.
Artist - User: Iedere artiest wordt door maar 1 user aangemaakt.

Artist - Song: Een artiest kan 0 of meerdere songs hebben.
Song - Artist: Iedere song hoort bij 1 artiest.
